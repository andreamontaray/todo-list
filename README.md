# To-Do List

A minimal and responsive to-do list app with dark mode, local storage, and intuitive item editing, completion, and deletion.

## 🔥 Features

- ✅ Add, edit, and delete tasks
- 🌙 Toggle dark mode
- 💾 Persistent with `localStorage`
- 📝 Click to toggle completed
